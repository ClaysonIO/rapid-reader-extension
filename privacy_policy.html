Rapid Reader - Privacy Policy

Overview

Rapid Reader is an open-source browser extension designed to help you read content faster using the RSVP (Rapid Serial Visual Presentation) reading technique. This privacy policy explains how we handle your data and privacy.

Data Collection

We do not collect any data from you. The Rapid Reader extension:

  • Does not collect, store, or transmit any personal information
  • Does not track your browsing history
  • Does not send any information to our servers or third parties
  • Does not use cookies or any other tracking technologies

How Your Data Is Used

The extension only processes the text you select on a webpage to display it using the RSVP reading method. This processing happens entirely within your browser, and the selected text never leaves your device.

Permissions

The extension requires the following permissions:

  • activeTab: To access the currently active tab to retrieve the text you've selected
  • scripting: To execute scripts that retrieve your selected text

These permissions are used solely for the functionality of the extension and not for data collection.

Open Source

Rapid Reader is open-source software. This means:

  • The complete source code is publicly available for review
  • Anyone can verify that the extension behaves as described in this privacy policy
  • The community can contribute to improving the extension

Changes to This Privacy Policy

If we make changes to this privacy policy, we will update the version in the extension and provide a link to view the changes.

Contact

If you have any questions about this privacy policy or the Rapid Reader extension, please open an issue on our GitHub repository.
